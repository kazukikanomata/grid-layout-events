テンプレート2: 複雑なグリッドレイアウト

体験ポイント: grid-column と grid-row でアイテムの配置を自由自在にコントロール！
試してみよう: 「.item1」の「grid-column: 1 / 3」を「1 / 4」に変更してみよう

.item1 { grid-column: 1 / 3; } /* 1列目から3列目の手前まで */
.item4 { grid-column: 2 / 4; grid-row: 2; } /* 配置を指定 */
.item6 { grid-column: 1 / 4; } /* 全列にまたがる */
チャレンジ: grid-template-rows の値を変更して、行の高さを調整してみよう！
